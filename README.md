# 🎓 Gestor de Estudos Acadêmicos Pro

**Projeto realizado por:** Renato Moreira Santos Faria
**Objetivo:** Auxiliar alunos com dificuldades de organização através de um cronograma de foco e gestão de tarefas.

---

## 🚀 Funcionalidades
- **Cadastro Dinâmico:** Registro de matérias e carga horária.
- **Persistência Local:** Uso de SQLite para manter os dados salvos entre sessões.
- **Controle de Status:** Marcação visual de tarefas pendentes ou concluídas.
- **Interface Intuitiva:** Desenvolvida em Tkinter para facilitar o uso.

## 🛠️ Qualidade e Automação
- **Linting:** Padronização PEP8 verificada com Flake8.
- **Testes Unitários:** Validação de lógica com Pytest.
- **CI/CD:** Pipeline automatizada via GitHub Actions.

---

## 📦 Como rodar a aplicação

Siga estas etapas para executar o projeto em sua máquina:

### 1. Pré-requisitos
Tenha o **Python 3.10 ou superior** instalado em seu sistema.

### 2. Clonar e Instalar
Abra o terminal e execute os comandos abaixo:
```bash
# Clonar o repositório
git clone [https://github.com/renasrenas2/organizador-estudos.git](https://github.com/renasrenas2/organizador-estudos.git)
cd organizador-estudos

# Instalar dependências de teste e qualidade
pip install -r requirements.txt
